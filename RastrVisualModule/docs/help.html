Программное обеспечение для работы с растровыми матрицами, Протопопов Михаил, 2016

Руководство пользователя

Начало работы

Чтобы начать работу с приложением, необходимо Загрузить, Импортировать или Создать растровую матрицу в памяти программы.
Для этого достаточно выбрать соответствующий пункт меню.

Загрузка

Чтобы загрузить файл достаточно найти его с помощью появившегося после нажатия на пункт меню окна и нажать кнопку ОК. На рабочем поле программы появится серое отображение растра.

Импортирование

Чтобы импортировать файл достаточно найти его с помощью появившегося после нажатия на пункт меню окна и нажать кнопку ОК. На рабочем поле программы появится серое отображение растра.

Создание нового растра

Чтобы создать новый растр нужно использовать пункт меню "Новый растр". Перед пользователем появится окно, в котором нужно будет ввести параметры создаваемого растра - количество элементов по ширине и по высоте.
Если в памяти программы уже есть растровая матрица, то при использовании одной из вышеперечисленных функций пользователю будет предложено сохранить текущую матрицу. Если пользователь согласится, то перед ним появится окно сохранения растров, а после завершения процедуры сохранения - окно загрузки. В противном случае на экране сразу появится окно загрузки/импорта/создания нового растра.

Работа с растрами

После того, как в память программы была введена растровая матрица, пользователю будет кнопка "Начать" станет доступной. С её помощью пользователь может задать параметры работы и отображения растровых матриц на рабочем поле программы.
При нажатии на кнопку "Начать" пользователь получит доступ к окну "Начало работы", в котором он может задать параметры осцилляции, режима работы и шага, с которого начнется работа.

Осцилляция

Определяет вид сканирующей матрицы, а также как она будет производить сканирование. Может быть включена и выключена.

Режим работы

Определяет режим работы с растрами в рабочей области программы. Ручной режим подразумевает, что у пользователя есть полный контроль над движением матрицы, а автоматический - что движение сканирующего растра будет производиться автоматически.

Начальный шаг работы

Задает шаг, с которого начинается движение растровой матрицы.
После задания параметров работы приложения, в рабочей области программы появится еще один растр, а также станет доступной для нажатия кнопка ">>", по нажатию на которую, пользователь запустит процесс движения сканирующий матрицы.
В процессе движения будут заполняться графики в нижней части экрана. Какие графики заполняются зависит от режима осцилляции: если он выключен, то заполняется только первый график, если включен - заполняются все 3 графика, в левый идут данные в нижнем положении сканирующего растра, в средний - в верхнем положении, в правый - разница между левым и средним.

Другие функции

По нажатию на кнопку "Цвет" в правой части экрана пользователь может изменить вид отображения рабочей области программы на более соответствующий реальной работе спектрометра.
